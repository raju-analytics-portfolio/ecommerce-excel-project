# ecommerce-excel-project
Excel Ecommerce Analytics is a simple project that analyzes online store data using Microsoft Excel. It focuses on data cleaning, sales analysis, product performance, customer insights, and creating an interactive dashboard.

Extracted and analyzed pizza sales data using SQL Server

Wrote SQL queries (SELECT, WHERE, GROUP BY, JOIN, HAVING) to answer business questions

Cleaned and transformed data for accurate analysis

Built an interactive Excel dashboard using:

Pivot Tables

Charts & KPIs

Sales trends and category-wise analysis
